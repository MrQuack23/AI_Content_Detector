Comparaison between the Models metrics

By leveraging the power of visualization, we aimed to comprehensively compare the efficacy of these models across each metric. We first extracted the relevant performance data from each model, encapsulated within pickle files. These metrics were then consolidated into a single dataset, enabling us to create insightful visual representations.
